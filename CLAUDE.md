# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 项目概述

"心域 - 内心是一片草地" — 一个纯前端个人成长与正念笔记应用，以"草地"为隐喻来可视化日常任务、习惯和情绪。

## 技术栈

- **纯 HTML/CSS/JavaScript** 单页应用，无框架、无构建工具
- 数据存储：浏览器 `localStorage`
- SVG 手绘风格图形（内联在 HTML 中）
- 依赖：Google Fonts (Noto Serif SC) — 外链字体

## 运行方式

直接在浏览器中打开 `index.html` 即可运行，无需任何服务器或构建步骤。

## 架构要点

### 数据模型 (`state` 对象，第 1470-1500 行)
所有应用状态集中保存在 `state` 对象中，通过 `localStorage` 读写：
- `tasks` — 任务列表（含类型分类、完成状态）
- `weeds` — 杂草（未完成的杂事可视化，3天后枯萎消失）
- `tree` — 成长树（多分支，每根树枝可长叶子/侧枝/增粗）
- `habits` — 今日习惯（good/bad/care 三组数组）
- `characters` — 角色状态（人鱼可见性、百合花开放度、黑猫冷静度、鹿友好度）
- `grass` — 草地自评等级
- `reviews` — 每日复盘列表
- `mirror` — 自我之镜的树形节点
- `history` — 每日快照历史记录

### 任务类型系统 (第 1462-1468 行)
5 种任务类型，通过关键词自动分类：
- `basic` → 生成杂草，完成后除草
- `study` → 完成时触发成长树
- `habit` → 完成时记入好习惯
- `free` → 完成后让人鱼出现
- `work` → 常规工作

### 角色交互（第 1888-1958 行）
4 个 SVG 角色响应玩家行为：
- **人鱼** — 完成自由类任务后出现
- **百合花** — 记录自我关怀后逐步绽放
- **黑猫** — 坏习惯或连续差评时烦躁，点击可安抚
- **梅花鹿** — 连续好习惯后变得友好

### 每日周期 (第 1524-1555 行)
每日首次加载时自动归档前一日数据到 `history`，并清空习惯/自评，任务保留不变。

### 页面结构
6 个页面通过底部导航切换（第 1433-1458 行），用 CSS class `active` 控制显示：
- 草地（主页：任务+自评）
- 习惯记录
- 每日复盘
- 自我之镜（树形思维导图）
- 打卡历史
- 设置与备份
